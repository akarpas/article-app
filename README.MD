# ARTICLES APP

### DEPLOYED
[View](https://articlesmag.netlify.com/)

### RUNNING LOCALLY

##### CLIENT

```yarn install && yarn start```

Client will run on localhost:8081

### TESTING

##### TEST
```yarn install && yarn test``` or ```yarn install && yarn test:watch```

##### TEST WITH COVERAGE
```yarn install && yarn test:cover```
